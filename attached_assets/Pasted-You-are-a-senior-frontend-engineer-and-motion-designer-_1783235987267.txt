You are a senior frontend engineer and motion designer specializing in award-winning, production-ready websites.

I want to upgrade my existing website's motion design to feel comparable in quality to websites like Myna Kitchen and Clay. Do not copy their designs, layouts, colors, typography, or branding. Instead, recreate the same feeling of smoothness, premium quality, depth, and polish while preserving my existing UI and brand identity.

Overall Motion Style

The experience should feel:

Premium
Modern
Cinematic
Smooth
Luxurious
Intentional
Fast
Confident

Every animation should have a purpose.

Avoid making the website feel playful or overly animated.

Hero Section

Make the hero feel alive.

Add:

layered entrance animations
staggered text reveal
subtle parallax depth
animated gradient lighting
gentle floating decorative elements
smooth CTA entrance
mouse-based movement with very small intensity
soft glow effects

Motion should be subtle enough that users notice quality without noticing "animation."

Scroll Experience

Every section should reveal naturally.

Add:

fade + translate animations
staggered reveals
progressive content appearance
image masking reveals
soft scale transitions
slight depth effects

Do NOT simply fade everything in.

Each section should have slightly different motion while maintaining consistency.

Cards

Upgrade every card.

Hover should include:

slight elevation
dynamic shadow
soft border glow
icon movement
tiny scaling (1.01–1.03)
subtle lighting effects

Cards should feel tangible.

Buttons

Make buttons feel premium.

Add:

magnetic hover
smooth background transition
subtle shine animation
soft scaling
elegant easing

Never use flashy effects.

Navigation

Upgrade the navbar.

Add:

glassmorphism while scrolling
smooth background blur
active page indicator
animated underline
elegant hover states
hide/reveal on scroll
Images

Add premium interactions:

slow zoom on hover
masked reveals
subtle parallax
smooth loading transitions
gentle shadow animation
Background

Introduce depth using:

animated gradients
soft radial glows
blurred shapes
subtle moving light
noise texture if appropriate

These should be extremely subtle.

Typography

Animate headings beautifully.

Use:

staggered line reveals
clipping masks
fade + translate
character or word animations only where appropriate

Body text should animate minimally.

Cursor Interactions

Add only tasteful cursor effects:

subtle glow
magnetic buttons
slight parallax

Do not replace the cursor.

Performance

The site must remain extremely smooth.

Target:

60 FPS
minimal layout shift
GPU-accelerated transforms
no janky scrolling
no unnecessary re-renders

Use transform and opacity instead of layout properties wherever possible.

Accessibility

Respect prefers-reduced-motion.

All animations should gracefully disable when users request reduced motion.

Technical Constraints
Do NOT redesign the website.
Do NOT modify existing branding.
Do NOT change spacing unless necessary for motion.
Do NOT change typography.
Do NOT break responsiveness.
Do NOT introduce hydration issues.
Keep components modular.
Avoid unnecessary dependencies.
If using Framer Motion or GSAP, integrate them cleanly and only where they add clear value.
Ensure the project builds successfully with npm run build.
Ensure the project deploys on Vercel without any configuration changes.
Fix any TypeScript or ESLint issues introduced.
Final Goal

When users land on the website, they should immediately feel:

"This company pays attention to every detail."

The animations should enhance professionalism and trust rather than distract from the content. The final experience should feel closer to a premium digital agency website than a standard portfolio, with motion that is smooth, refined, and production-ready.
